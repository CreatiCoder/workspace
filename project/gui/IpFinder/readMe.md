# IP Finder

### 1. 개요

#### 자바 GUI 라이브러리 AWT을 기반으로 작성된 프로그램입니다. 

#### 도메인(ex. www.google.com )을 입력하여 아이피(IP)를 출력하며 로그 화면에 그 결과가 보여집니다.
