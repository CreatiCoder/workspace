# Image Widget

### Overview

자바 스윙을 이용해 만든 gif 움짤 뷰어



### Usage

##### 1. 실행 후 gif 파일을 프로그램 내부로 드래그 해주면 됩니다.

##### 2. 드래그 하여 위치를 이동시 킬 수 있습니다.

#####3. 더블 클릭하여 종료할 수 있습니다.

##### 4. 다른 gif를 다시 드래그하면 해당 gif파일로 바뀝니다.

##### 5. gif의 속도나 크기는 원본을 따라갑니다. 크기와 속도를 바꾸고 싶다면 원본 gif를 수정하세요.
